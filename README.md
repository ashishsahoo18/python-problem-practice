# Python Problem Practice 🐍

A collection of Python practice problems designed to improve programming logic, problem-solving skills, and Python fundamentals.

## About Project

This repository contains Python solutions for different programming problems. 
It covers basic to intermediate level questions that help in building strong coding concepts.

## Topics Covered

- Python Basics
- Variables and Data Types
- Conditional Statements (if-else)
- Loops (for, while)
- Functions
- Lists and Tuples
- Strings
- Dictionaries
- Sets
- Recursion
- Pattern Problems
- Searching & Sorting
- File Handling
- Object-Oriented Programming (OOP)
